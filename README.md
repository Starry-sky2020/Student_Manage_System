# Student_Manage_System

## JAVASE 学生管理系统
分为用户和管理员两个角色

环境：  
开发工具IDEA   
JDK1.8    
apache-maven-3.6.8 （用来包管理）      


项目内容：学生管理系统  
1）	纯命令行（不需要画web页面）  
2）	用到基础知识  
---java的流程控制、面向对象基础知识、集合、文件等（多线程、网络编程不涉及）
